This project develops a generative decision-making industrial large model specifically designed for manufacturing production scenarios. The model integrates time series foundation models with domain-specific engineering process knowledge, enabling intelligent decision support for production scheduling, quality control, and resource optimization. As a core technical member, I contribute to the model architecture design and the integration of AI engineering methodologies into the development pipeline.
